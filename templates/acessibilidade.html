Acessibilidade

Nosso Compromisso

A UNIVESP está comprometida em tornar seu site de polos acessível a todos os usuários, independentemente de suas capacidades ou tecnologias assistivas utilizadas.

Recursos de Acessibilidade

Navegação por Teclado

  • Tab: Navega entre elementos interativos
  • Enter/Space: Ativa o elemento selecionado
  • Alt + 1: Ir para página inicial
  • Alt + 2: Ir para busca de polos
  • Alt + 3: Ir para lista de polos
  • Alt + 0: Ir para página de acessibilidade

Alto Contraste

Você pode ativar o modo de alto contraste clicando no botão "Alto Contraste" no rodapé da página ou usando o atalho Alt + C.

Redimensionamento de Texto

Para aumentar ou diminuir o tamanho do texto:

  • Pressione Ctrl + (+) para aumentar
  • Pressione Ctrl + (-) para diminuir
  • Pressione Ctrl + 0 para restaurar o tamanho padrão

Tecnologias Assistivas

Nosso site é compatível com os principais leitores de tela, incluindo:

  • NVDA
  • JAWS
  • VoiceOver
  • TalkBack

Mapa Acessível

Para usuários que não podem utilizar o mapa visual, oferecemos:

  • Lista textual de polos com distâncias
  • Descrições detalhadas de localização
  • Navegação por teclado no mapa

Feedback e Suporte

Se você encontrar alguma dificuldade de acesso ou tiver sugestões para melhorar nossa acessibilidade, entre em contato conosco:
